# Model

В данном репозитории представлен программный код, который был написан в бакалавриате, в рамках моей выпускной квалификационной работы.
<br> <strong> Тема ВКР </strong> - Программный комплекс для имитационного моделирования удалённого беспроводного доступа пользователя к распределённым вычислительным ресурсам
#### Год защиты работы - 2021 <br>
### Основная идея представленной работы 
Очень скоро IoT настолько плотно укоренится в нашей жизни, что станет неотъемлемой её частью. Таким образом, количество устройств, с возможностью передачи данных, будет только увеличиваться. Следовательно, и нагрузка на мобильные сети передачи информации будет так же возрастать. Для того, чтобы частично снизить нагрузку на сети, в их состав, начиная с пятого поколения были добавлены “MEC” (Multi-Access Edge Computing) сервера. Задачей данных серверов, стала обработка части задач “на месте”, т.е. они вычисляют задачи без передачи данных на облачные сервера. Работа данного типа серверов должна снизить нагрузку на сети, так как пропадёт необходимость дополнительной передачи части данных для последующей их обработки, а это должно положительно сказаться на задержке и выходной интенсивности системы. Именно для “MEC” серверов и возникает задача, как грамотно распределить предоставляемые ими вычислительные мощности. Например, в случаях, когда устройства могут перемещаться между различными областями сети, а следовательно, и оказываться в зоне работы разных серверов. Поэтому необходимо понять, как максимально эффективно организовать нагрузку на сервера. Когда пользователю лучше переносить свою задачу на другой сервер, а когда оставлять на текущем. Для того, чтобы получить аналитические данные, которые бы позволили оценить эффективность работы системы необходимо произвести имитационное моделирование работы системы. Поэтому целью данной работы является разработка программного комплекса для имитационного моделирования доступа пользователя к вычислительным ресурсам сотовой сети, в составе которой присутствуют сервера, расположенные рядом с базовой станцией.
